# 第 2 章論理

## 含意 A ならば B

あまり一般的ではなかった。特に A が false の場合。
A が false の場合は B はなんでも良いという意味の「ならば」が該当する

| A     | B     | A ⇒ B |
| ----- | ----- | ----- |
| true  | true  | true  |
| true  | false | false |
| false | true  | true  |
| false | false | true  |
